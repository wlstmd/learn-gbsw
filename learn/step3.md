## 수식 `a*2+(b+c)`를 전위 표기법으로 나타내면?

```
1. 가장 바깥쪽 연산자부터 처리
- a*2+(b+c) → +(a*2)(b+c)

2. 왼쪽 부분식 a*2 처리
- a*2 → *a2

3. 오른쪽 부분식 (b+c) 처리
- (b+c) → +bc

4. 최종적으로 합치기
- +(a*2)(b+c) → +*a2+bc
```

답 : `+*a2+bc`

## 수식 `a*2+(b+c)`를 후위 표기법으로 나타내면?

```
1. 괄호 안부터 처리
- (b+c) → b c +

2. 그 다음 `a*2 부분`을 처리
- a*2 → a 2 *

3. 마지막으로 전체를 더하는 연산을 처리
- (a*2) + (b+c) → a 2 * b c + +
```

답 : `a2*bc++`
